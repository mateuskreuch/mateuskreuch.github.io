Predicting factorization exponents

While helping a friend with a job interview and solving some optimization problems, I stumbled upon a really cool little recurrence that predicts the exponents of multiples of a prime. The problem that led to it can be read here and here.

The recurrence itself is this:

ε ( n , f + 1 ) = ε ( n , f ) ⌢ ... ⌢ ε ( n , f ) ⏞ n times and add 1 to its last element
ε ( n , 0 ) = ( 1 )

So, for example, if we want to find the exponents of two for the factorization of the first eight multiples of two:

 ε(2, 0) = (1)

 ε(2, 1) = (1, 1 + 1) = (1, 2)

 ε(2, 2) = (1, 2, 1, 2 + 1) = (1, 2, 1, 3)

 ε(2, 3) = (1, 2, 1, 3, 1, 2, 1, 3 + 1) = (1, 2, 1, 3, 1, 2, 1, 4)

Which predicts the following factorizations:

  2 = 2^1

  4 = 2^2

  6 = 2^1 * 3^1

  8 = 2^3

 10 = 2^1 * 5^1

 12 = 2^2 * 3^1

 14 = 2^1 * 7^1

 16 = 2^4
